Algorithms and Calculations

Customer Lifetime Value (LTV)

// LTV Algorithm
function calculateLTV(customer) {
  const avgBooking = customer.totalRevenue / customer.bookingCount;
  const frequency = customer.bookingCount / monthsActive;
  const lifespan = customer.loyaltyTier === 'Platinum' ? 5 : 3;
  return avgBooking * frequency * 0.18 * lifespan;
}
